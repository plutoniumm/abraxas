Modifiers

Since only basic gates are supported, additional gates can be added via hooks into the toPrime function.

Qiskit

Hooks accepts a dictionary with the gate name as the key. The value can be a single function or a list of functions to be applied to the circuit. In the example above the x\sqrt xx​ gate is substituted by a RxR_xRx​ and a PPP gate.

Each function receives the quantum circuit and the gate as arguments. The gate is a tuple with the gate name and the qubits it acts on.
